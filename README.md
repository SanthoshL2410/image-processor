## Project Description

This project is an Image Processor Tool developed using Python. It helps users perform basic image processing operations such as opening, resizing, rotating, and saving images through a simple Python program.

The program reads images from the `input_images` folder and stores the processed images in the `output_images` folder.

## Features

* Image Loading

  * Open image from folder
  * Read image formats (JPG, PNG, etc.)

* Image Processing

  * Resize image
  * Rotate image
  * Crop image
  * Apply filters (if implemented)

* Image Saving

  * Save processed image
  * Store output in `output_images` folder

## Technologies Used

* Python
* Pillow (PIL) Library

* Python source code

## How to Run

1. Open the project folder in VS Code or any Python IDE
2. Install required library:

```bash
pip install pillow
```

3. Place an image inside the `input_images` folder
4. Run the Python file
5. View the processed image in the `output_images` folder
